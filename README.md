# x-ui
支持多协议多用户的 xray 面板

# 功能介绍
- 系统状态监控
- 支持多用户多协议，网页可视化操作
- 支持的协议：vmess、vless、trojan、shadowsocks、dokodemo-door、socks、http
- 支持配置更多传输配置
- 账号流量统计
- 可自定义 xray 配置模板
- 支持 https 访问面板（自备域名 + ssl 证书）
- 更多高级配置项，详见面板

# 安装&升级
## 测试版
```
bash <(curl -Ls https://raw.githubusercontent.com/sprov065/x-ui/master/install.sh) 0.0.2
```

## 建议系统
- CentOS 7+
- Ubuntu 16+
- Debian 8+

# 常见问题
## 与 v2-ui 关系
x-ui 相当于 v2-ui 的加强版，未来会加入更多功能，待 x-ui 功能稳定后，v2-ui 将不再提供更新

x-ui 可与 v2-ui 并存，数据不互通，不影响对方的运行

## 从 v2-ui 迁移
将提供便捷的一键迁移方式，正在开发中
